# week-6-interactive-sales-dashboard
Interactive Sales Dashboard (Week 6)

Overview

This project builds an interactive sales dashboard using Python. It analyzes sales data and shows insights using different charts.

⸻

Tools Used
	•	Python
	•	Pandas
	•	Matplotlib
	•	Seaborn
	•	Plotly
	•	ImageIO

⸻

Dataset

The dataset contains sales details such as:
	•	Date
	•	Product
	•	Quantity
	•	Price
	•	Customer_ID
	•	Region
	•	Total_Sales

⸻

Features
	•	Sales by Product (Bar Chart)
	•	Sales Trend (Line Chart)
	•	Price Distribution (Box Plot)
	•	Correlation Heatmap
	•	Dashboard with multiple charts
	•	Interactive chart using Plotly
	•	GIF demo of dashboard

⸻

Output
	•	Visual charts displayed on screen
	•	GIF demo file included

⸻

Conclusion

This project shows how to analyze and visualize sales data using Python and build a simple interactive dashboard.
